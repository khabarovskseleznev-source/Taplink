=== ИНСТРУКЦИЯ ДЛЯ MY TAPLINK ===

1. ОТКРЫТЬ ПРОЕКТ В VS CODE
   - Запустите Visual Studio Code.
   - Меню "Файл" -> "Открыть папку" -> выберите папку "my_taplink".

2. НАСТРОИТЬ ПОД СЕБЯ (ОБЯЗАТЕЛЬНО!)
   В файле index.html замените:
   - [Ваше Имя и Фамилия] на ваше настоящее имя.
   - https://t.me/ВАШ_ЛИЧНЫЙ_TG на ссылку вашего личного Telegram.
   - https://t.me/ВАШ_BTC_TG на ссылку Telegram про Bitcoin.
   - https://t.me/ВАШ_СЕМЕЙНЫЙ_TG на ссылку семейного Telegram.
   - ВАША@ПОЧТА.ru на ваш email для сотрудничества.
   - [Ваше Имя] в подвале (footer) на ваше имя.

3. ЗАМЕНИТЬ АВАТАР
   - Замените ссылку в строке (ищете "via.placeholder.com") на прямую ссылку на ваше фото в интернете.
   - Или оставьте как есть и кликните на аватар прямо на странице, чтобы вставить новую ссылку.

4. ЗАПУСТИТЬ СТРАНИЦУ
   - В VS Code нажмите правой кнопкой на файл index.html -> "Copy Path".
   - Вставьте этот путь в адресную строку браузера.
   - Или просто откройте файл index.html через проводник двойным кликом.

5. ДАЛЬНЕЙШИЕ ШАГИ
   - Когда создадите YouTube, замените "#" в соответствующей кнопке на реальную ссылку.
   - Чтобы выложить страницу в сеть, используйте бесплатный хостинг GitHub Pages.
     (Инструкцию найдете по запросу "GitHub Pages hosting").
   - Полученную ссылку вставьте в Instagram.

УДАЧИ! Страница уже имеет современный дизайн и адаптирована под мобильные.
